lazyResponsive.js

A light weight javascript library to implement lazyloading with responsive download.

demo

include <script src="http://balajisoundar.github.io/javascripts/lazyResponsive.js" ></script> to your code

How to

create img element with an id and src element empty or anything (like a default image) .

<img id="logo" src="loading.gif" />

and in your javascript call 

lazyResponsive.load(config);
pseudo code for config obj is

{
	"min:max":{
		id:"url of the image"
	}
	.
	.
	.
	.
	.
	default:{
	   id:"url of the image"
	}
}

example:

var config={
	"770:1380":{
			logo:"http://example.com/logo.png"
	}
	"300:769":{
			logo:"http://example.com/logo_small.png"
	}
}

lazyResponsive.load(config);
